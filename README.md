## Featured Projects

### [ATLAS](https://github.com/atoutalligotr/atlas)
> Actuarial exam study platform with spaced repetition, daily games, and a full content management system.

Built for students preparing for actuarial exams. Covers everything from structured topic content and flashcards to practice problems, forum discussions, and a character/XP progression system. Admins manage all content through an in-app block editor.

**Stack:** Next.js 16 · React 19 · TypeScript · PostgreSQL · Prisma · NextAuth · Tailwind CSS v4

---

### [Excel Delta View](https://github.com/atoutalligotr/excel-delta-view)
> VS Code extension for side-by-side Excel file comparison with cell-level diffs and exportable reports.

Pick two `.xlsx` files, select sheets to compare, and get a visual diff with similarity scores per sheet. Export results as a multi-tab Excel report or a self-contained, password-protected HTML file (AES-256-GCM).

**Stack:** TypeScript · VS Code API · ExcelJS · esbuild

---

### [Flash Layer](https://github.com/atoutalligotr/flash-layer)
> Offline-first flashcard app with SM-2 spaced repetition — no account, no server, all data lives in your browser.

Anki-style 4-option ratings, JSON deck import, course grouping, streak tracking with a 90-day heatmap, and 6 card themes. Installable as a PWA or packageable as a desktop app via Tauri.

**Stack:** React 19 · TypeScript · Vite · Zustand · Dexie.js (IndexedDB) · Framer Motion · Tailwind CSS v3

---

### [XML Deep View](https://github.com/atoutalligotr/xml-deep-view)
> VS Code extension that renders large XML and JSON files as collapsible, searchable tree views.

Handles files that break VS Code's built-in formatter. Color-coded nodes, inline search with auto-expand, and a stats panel showing node counts, max depth, and tag frequency — all via a SAX streaming parser for large file support.

**Stack:** TypeScript · VS Code API · fast-xml-parser · esbuild
